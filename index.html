FAQ

常见问题

Agent 是真的在替我交易吗？▾
是的。Agent 通过 Claude API 大脑做决策，调用 1024EX 和其他平台的 API 执行交易。所有操作链上可查。你可以设置为全自动模式或半自动（大额需确认）模式。
Agent 会不会亏光我的钱？▾
Agent 内置多层风控：每日亏损上限、单笔最大仓位、最大杠杆限制、最大回撤止损。你可以自定义所有风控参数。但任何交易都有风险，建议只投入可承受损失的资金。
什么是 Agent 基因？跟 AI 有什么关系？▾
Agent 基因是 16 个 0.0-1.0 的数值，分布在 4 条染色体上（风险性格、市场直觉、资产偏好、智能风格）。它们决定 AI Agent 的交易决策风格 — AI（Claude API）是 Agent 的大脑，基因是 AI 决策的"性格参数"。
需要多少钱才能开始？▾
孵化 Agent 免费。交易最低资金取决于策略 — 稳定币收益（Atlas）最低 50 USDC，杠杆动量（Titan）建议 200+ USDC。繁殖需要 5 USDC + 24h 冷却。可先用模拟模式体验。
繁殖出来的后代真的更强吗？▾
不保证，但概率上有优势。通过遗传算法（交叉 + 突变），后代可能继承双亲优势基因。0.5% 传奇突变概率可能诞生超越父母的变异体。也可能更弱 — 这就是进化的魅力。
2.5% 后代分润怎么算？▾
后代 Agent 产生利润时，2.5% 归双亲主人平分（各 1.25%）。这是永久被动收入。亏损时不收。你的 Agent 繁殖出越多成功后代，被动收入越多。
需要会编程吗？▾
完全不需要。说人话就行 — "我有 500U 想日赚 50U" 就是一条完整的指令。如果你是开发者，也提供完整的 API + Agent SDK。
去黑盒具体是什么意思？▾
每一次 Agent 决策都有完整记录：用了什么数据源、为什么得出判断、为什么选择这个仓位、为什么在这个时间点执行。全部透明可查，路径可追踪，结果可复盘。没有"神秘算法"。
System Owner 是什么？▾
System Owner 不是"用户"，是这个 Agent 交易系统的主人。你部署 Agent、定义基因、繁殖后代、在策略商店上架。首批 1,024 个名额，申请制进入。
