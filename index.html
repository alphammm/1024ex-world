FAQ

常见问题

1024EX 和其他交易所有什么不同？▾
1024EX 是全栈链上交易所，同时支持永续合约、预测市场、大宗商品，并内置 AI Agent 系统。所有交易链上执行，Gas 费约 $0.01。Agent 拥有 DNA 基因，可繁殖进化。
预测市场怎么玩？▾
在 1024EX.com 选择一个预测事件，买入 YES 或 NO。如果事件发生，YES = $1；不发生，NO = $1。链上结算，公平透明。也可以让 Agent 自动扫描跨平台套利机会。
Agent 是真的在替我交易吗？▾
是的。Agent 通过 Claude API 做决策，调用 1024EX API 执行交易。所有操作链上可查、去黑盒。你可以设置全自动或半自动模式。
需要多少钱才能开始？▾
预测市场最低 1 USDC 起。孵化 Agent 免费。交易最低资金取决于策略 — 稳定币策略最低 50 USDC。可先用模拟模式体验。
什么是 Agent DNA？▾
16 个 0.0-1.0 的数值分布在 4 条染色体上（风险性格、市场直觉、资产偏好、智能风格），决定 Agent 的交易决策风格。不同 DNA = 完全不同的交易行为。
需要 KYC 吗？▾
不需要。连接钱包即可使用全部功能。支持 MetaMask、WalletConnect、Coinbase Wallet、邮件登录（Privy）。
System Owner 是什么？▾
不是"用户"，是系统的主人。你部署 Agent、定义基因、繁殖后代、在策略商店上架。首批 1,024 个名额，邀请码进入。
